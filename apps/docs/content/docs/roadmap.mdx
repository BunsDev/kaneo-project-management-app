This roadmap outlines our planned features and improvements for Kaneo. While we're committed to delivering these enhancements, the timeline and priorities may shift based on community feedback.

<Callout type="info">
  We welcome community input on our roadmap! Your feedback helps us prioritize development efforts.
</Callout>

## Core Task Management

<Accordions type="multiple">
  <Accordion title="Task Organization">
    - Task deletion capability
    - Subtasks/child issues
    - Linking tasks (parent/child relationships)
    - Move tasks between projects
  </Accordion>

  <Accordion title="Task Creation">
    - Quick task creation (text box in each column)
    - Default priority set to "normal"
    - Option to disable WYSIWYG editor
    - Fix priority selector on mobile
  </Accordion>
</Accordions>

## Content & Attachments

- File attachments for cards
- Image uploads with cover images
- Due dates with configurable defaults
- Reminders

## Board Customization

<Accordions type="multiple">
  <Accordion title="Layout & Organization">
    - Custom pipelines/columns
    - Editable statuses
    - Custom fields configuration
    - Tags/labeling system
  </Accordion>

  <Accordion title="Views & Sorting">
    - Swimlanes (by team member, date, priority)
    - Sorting options within columns
  </Accordion>
</Accordions>

## Integration & Authentication

- GitHub integration
- API access for automation
- OIDC and SAML support
- PostgreSQL support

## Advanced Features

- Gantt chart view
- Task dependencies
- Time tracking data
- Global search functionality

## Team Collaboration

- Role-based access (Admin, Collaborator, Viewer)
- Granular permissions for boards/tasks
- Email and push notifications
- Activity tracking
